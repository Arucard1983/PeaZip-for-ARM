EXTRAIR DE ARQUIVO (S)
A partir do sistema
- Rightclick sobre o arquivo (s) e no menu de contexto do sistema, clique em "Extrair aqui" ou "Extrair aqui (em nova pasta)" para extrair, sem interação mais
- Utilizar de outra forma "Extrair ..." entrada de menu para ter mais opções: saída caminho, senha, extrair os arquivos para um novo diretório, preferiu pular, renomear ou substituir arquivos existentes etc

Abrir um arquivo em PeaZip, ou seja, com a DoubleClick, ou arraste um arquivo na janela ou ícone PeaZip
- Clique em "Extract" na barra de ferramentas ou no menu de contexto para extrair apenas o selecionado através de objetos o diálogo de confirmação que permite definir todas as opções (saída caminho, senha, nomeando política, extrair de um novo diretório etc)
- Arraste os arquivos e pastas para o destino desejado, apenas o conteúdo selecionado será extraído (para mais informações consulte o capítulo sobre arrastar e soltar no PeaZip)
- Clique no botão de extração rápida suspensa à direita de "Extract" na barra de ferramentas, que permite extrair diretamente o arquivo completo para os destinos mais comuns (ignorando o diálogo de confirmação) e para definir as opções mais importantes; atalhos de teclado: Ctrl + E ( ou F12) extrai todo o arquivo, pede diretório de saída; Ctrl + Alt + E arquivo extratos em sua pasta atual; Ctrl + Shift + E para extratos desktop; Ctrl + Alt + Shift + E extratos aos documentos; Ctrl 1 .. 8 extratos a caminho do marcador 1 .. 8 (se definido)

PeaZip aberta, selecione uma ou mais arquivos e clique em "Extract" na barra de ferramentas ou menu de contexto, ou usar destinos extração rápidas, como explicado no ponto anterior.

Extrair apenas objetos selecionados a partir do arquivo
Abrir um arquivo em PeaZip, ou seja, com a DoubleClick, ou arraste um arquivo na janela ou ícone PeaZip
- Clique em "Extract" na barra de ferramentas: apenas os itens selecionados serão extraídos
- Arraste os arquivos e pastas para o destino desejado, apenas o conteúdo selecionado será extraído
- Rightclick e em "Extract" grupo do menu de contexto, clique em "Extrato selecionado" (para extrair apenas os itens selecionados) ou "extrair exibida" (para extrair o conteúdo do diretório corrente ou de filtro de pesquisa atual)

Digite uma senha para navegar ou Extrair um arquivo
Clique no ícone de cadeado para entrar com uma senha e, opcionalmente, um keyfile.
O ícone é destaque tanto na barra do navegador PeaZip do arquivo de status e na interface de extração do arquivo, uma vez que a senha é definida, o ícone muda de cor.

CRIAR ARQUIVO
A partir do sistema
- Rightclick em objetos para ser arquivado e clique em "Adicionar ao arquivo" no menu de contexto ou no menu SendTo. Ele vai abrir o arquivo de diálogo de confirmação de criação, há mais opções disponíveis na aba "Avançado", clique em OK para criar o arquivo.
- Em alternativa, arrastar os arquivos / pastas a janela PeaZip ou atalho, a interface de criação mesmo arquivo será mostrado.

De PeaZip
- Selecionar os objetos a serem arquivados e clique no botão "Adicionar", a interface de criação mesmo arquivo será mostrado.

Incluir arquivos / pastas PARA ATUALIZAR arquivo existente
Abrir um arquivo em PeaZip (ou seja, com a DoubleClick, ou arraste um arquivo na janela PeaZip ou ícone), em seguida, arrastar arquivos e pastas a serem adicionados no arquivo (ou clique no botão "Adicionar" e menu do aplicativo uso do contexto para adicionar objetos ao arquivo).
Ele vai abrir o arquivo de diálogo de confirmação de criação, clique em OK para atualizar o arquivo existente.

CRIAR SPLIT ARQUIVO em arquivos menores
Embora a criação de um arquivo como explicado nos pontos anteriores, clique em "Single volume" menu suspenso para selecionar um tamanho para os arquivos de saída (volumes) o arquivo será dividido dentro
Tipos mais comuns de arquivos suporta essa opção.

CRIAR arquivo criptografado
Clique no ícone de cadeado para entrar com uma senha e, opcionalmente, um keyfile, o ícone é destaque tanto na barra do navegador PeaZip do arquivo de status e na interface de criação de arquivo.
Para ocultar nomes de arquivos e diretórios contidos no arquivo verificar "Criptografar também nomes de arquivos", por favor, note que ele será aplicado somente se o arquivo será criado em um formato que suporte esta função, como 7Z e ARC.
Note que no arquivo interface de criação, ao lado do ícone de cadeado, há um aviso para informar visualmente se a criptografia está definido e se o formato de arquivo atual suporta criptografia.

CRIAR arquivos separados
Adicionar objetos a serem arquivados (com PeaZip botão "Adicionar", ou a partir do menu de contexto do sistema ou menu SendTo) e antes de confirmar com cheque "Ok" "Adicionar cada objeto para um arquivo separado" opção.

Converter arquivos
De PeaZip selecione os arquivos a serem convertidos e clique em "Convert" na barra de ferramentas ou menu de contexto, arquivamento de arquivos e pastas não podem ser adicionados, bem como, com a diferença de em arquivos sendo extraídos antes do estágio de compressão.
Usando o "Converter arquivos existentes" opção em conjunto com "Adicionar cada objeto para um arquivo separado" (padrão) que realiza uma conversão em massa de arquivos listados, sem que a conversão de arquivo chave é para consolidar dados de entrada em um único arquivo, melhorando a compressão eficiência, pois permite recompressão dos dados originais de sua forma descompactada.

DIRETAMENTE criar arquivos em um formato especificado e com um nível de COMPRESSÃO ESCOLHIDO
Usando o "sistema de integração" procedimento no menu Opções é possível habilitar entradas do menu de contexto para adicionar diretamente os arquivos / pastas selecionados para um arquivo ZIP 7Z, ou auto-extraível.
Para os formatos ZIP e 7Z também é possível para permitir entradas do menu de contexto para comprimir o nível mais rápido, normal ou ultra, ultrapassando o nível de compressão padrão para o formato selecionado.

CRIAR ARQUIVOS

1) Adicionar itens a um layout de arquivo a partir do menu contexto do sistema e menu "Enviar para", pelo arrastar e soltar ou botões e menus da aplicação;
2) Clique em "Criar" para iniciar a criação de arquivo (caso contrário, você pode salvar o layout do arquivo para uso futuro).

Uso:
- Aba "Opções" ara personalizar o formato de arquivo e compressão e encriptação de parâmetros relacionados; 
- Aba "I/O" para as opções de entrada / saída;
- Aba "Filtros" para a utilização de filtros de exclusão avançadas, se disponível para o formato de arquivo escolhido;
- Aba "terminal"  para análise e personalização da sintaxe da tarefa, ou exportar a tarefa como linha de comando.

Alternativamente, em uma única passagem você pode adicionar cada item de entrada para um arquivo separado pelo menu de contexto do sistema, e no menu "Enviar para" você pode adicionar vários itens para um único arquivo.

No layout de arquivo que você pode também:
- Duplo-clique em um objeto para abri-lo
- Selecione um arquivo e pressione "\" ou "ferramentas de arquivo> dividir arquivo" para dividi-los.
- Selecione um arquivo e pressione "<" ou "ferramentas de arquivo> Compare arquivos" byte a byte para compará-lo com outro arquivo.
- Selecione vários objetos e pressione "?" ou "Ferramentas de arquivo File> Checksum / hash do arquivo(s)" para checar a integridade dos arquivos
- Selecione vários objetos e pressione "*" ou "Apagar Seguro (arquivos)" para exclusão de arquivos de forma segura


ABRIR/EXTRAIR ARQUIVOS

1) Abra o(s) arquivo(s) pelo menu contexto e/ou pelo menu "Enviar Para", ou para associação do arquivo, ou selecione um ou mais arquivos pelo navegador do Peazip.
2a) Clique em "Extrair tudo" para extrair o arquivo, ou "Extrair tudo para" para ser perguntado sobre o caminho de saída;
2b) Clique duas vezes ou pressione "Enter" sob um arquivo para ver ele com o aplicativo associado, ou abra ele no Peazip se for um arquivo comprimido suportado, ou explore ele se for um diretório;
2c) Selecione os objetos arquivados e pressione o botão do meio do mouse ou a barra de espaço para extraí-los (para ser perguntado sobre o caminho de saída);
2d) Clique com botão direito do mouse no navegador do Peazip para usar itens do menu de contexto para executar todas as funções disponíveis, como extrair arquivos selecionados ou exibidos, prever, testar, listar, adicionar ou apagar arquivos etc...
Dica: se você quiser extrair um arquivo para um novo diretório, use o comando "Extrair em uma nova pasta" no menu contexto (ou aba I/O ), ou pressione \ ou Ctrl+N enquanto no navegador de arquivo.

Uso:
- Aba "Opções" para personalizar o formato com parâmetros específicos;
- Aba "I/O" para entrada / saída de opções, incluindo o comando "Extrair em uma nova pasta" (disponível também no menu de contexto);
- Aba "Filtros" para a utilização de filtros avançados de inclusão e exclusão, se disponível para o formato de arquivo;
- Aba "terminal" para análise da sintaxe e personalização de tarefa, ou exportação da tarefa como a linha de comando.

Você pode extrair vários arquivos de uma só vez a partir do menu de contexto do sistema ou a partir do navegador PeaZip.

Tipos de arquivos desconhecidos do PeaZip podem ser suportados (para criação e extração) através de um executável apropriado personalizado, selecionando formato "Personalizado" na aba "Opções".

SOBRE PEAZIP

PeaZip é um aplicativo de código aberto e gerenciador de arquivos: multi-plataforma, disponível como software portátil e instaláveis para 32 e 64 bits do Windows (9x, 2k, XP, Vista) e Linux (PeaZip é uma aplicação desktop neutro).PeaZip is an open source file and archive manager: cross platform, available as portable and installable software for 32 and 64 bit Windows (9x, 2k, XP, Vista) and Linux (PeaZip is a desktop neutral application).

PeaZip permite aplicar vários poderosos filtros de pesquisa para o conteúdo do arquivo, criar e extrair arquivos de uma só vez, para criar arquivos auto-extraíveis, a definição de exportação trabalho como linha de comando; salvar layouts arquivar; arquivos favoritos e pastas; digitalizar e abrir com aplicações personalizadas comprimido e arquivos descompactados etc ..

Outros Recursos: Forte criptografia, copia de arquivos robusta, dividir/juntar arquivos (file span), apagar seguro de dados, comparar, checksum e hash de arquivos, benchmark do sistema, geração randômica de senhas e arquivos-chaves.
